# BedandBreakfastGUI (console demo)

This is a minimal C# console application that demonstrates using `System.Console` and `Console.WriteLine()`.

How to run:

1. Ensure you have the .NET SDK installed (recommended: .NET 8.0).
2. From the repository root:

```bash
cd BedandBreakfastGUI
dotnet run
```

The program will print a few lines and wait for Enter.
